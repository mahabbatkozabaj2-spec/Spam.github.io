# 🛡️ SpamGuard — Детектор спама и мошенников

AI-анализатор для определения спама, фишинга и мошенников. Поддерживает **русский** и **казахский** языки.

## 🚀 Запуск на GitHub Pages (статический сайт)

Сайт работает **полностью в браузере** — без сервера. Достаточно включить GitHub Pages:

1. Загрузите все файлы на GitHub
2. Перейдите: `Settings → Pages → Source: Deploy from a branch → main / (root)`
3. Сайт будет доступен по адресу: `https://<ваш-логин>.github.io/<репозиторий>/`

### Файлы проекта

```
├── index.html      — главная страница
├── style.css       — стили
├── script.js       — логика интерфейса
├── analyzer.js     — движок анализа (работает в браузере)
├── _config.yml     — настройка GitHub Pages
├── app.py          — Flask сервер (только для локальной работы)
├── analyzer.py     — Python версия анализатора (для сервера)
└── requirements.txt
```

## 💻 Локальный запуск (с Python сервером)

```bash
pip install -r requirements.txt
python app.py
# Откройте: http://localhost:8000
```

> **Примечание:** Python сервер нужен только для анализа `.doc`, `.docx`, `.pdf` файлов.  
> Для `.txt`, `.eml`, `.html`, `.csv` — всё работает без сервера.

## ✨ Возможности

- 🔗 Анализ URL / ссылок — фишинг, тайпосквоттинг, подозрительные домены
- 💬 Анализ сообщений — SMS, WhatsApp, Telegram, Email на русском и казахском
- 📄 Анализ документов — текстовые файлы, письма (.eml), HTML
- 📊 Многоуровневая оценка риска: Фишинг, Мошенничество, Спам, Манипуляция
- 🇰🇿 Полная поддержка казахского языка

## 🔍 Что обнаруживает

- Фишинговые сайты и тайпосквоттинг (имитация брендов)
- Скам-вакансии с депозитом
- Финансовые мошенничества и крипто-скамы
- Запросы личных данных (карты, CVV, PIN)
- Поддельные письма от банков и госорганов
- Романтический скам
- Шантаж и вымогательство
- Манипулятивные приёмы (срочность, давление, ложные гарантии)

## 📱 Поддерживаемые форматы файлов (браузер)

`.txt` `.eml` `.html` `.htm` `.csv` `.log`
